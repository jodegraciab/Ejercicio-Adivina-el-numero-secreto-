Este es un programa interactivo desarrollado en Python de la asignatura Programación 3. 

El objetivo es que el usuario adivine un numero generado aleatoriamente, porbando intentos limitados y diferentes niveles de dificultad.

Permite elegir entre niveles que cambian el rango del numero (Facil, Medio, Dificil).

El jugador cuenta con un maximo de 5 intentos por partida.

El sistema almacena todos los numeros ingresados durante el juego.

Utilice  diccionarios para manejar opciones no validas por defecto.
